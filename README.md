# cloud-kora云端转山 - GitHub Pages 部署包

📦 包含文件：
- index.html - 网站主页
- success.html - 提交成功页面
- 1-快速开始.txt - 3步部署指南（先看这个！）
- 2-详细指南.md - 完整部署文档

⚡ 快速部署（10分钟）：
1. 创建 GitHub 仓库（公开）
2. 上传 index.html 和 success.html
3. 启用 GitHub Pages
4. 配置 Formspree 表单服务

📌 重要提醒：
- 必须修改 index.html 中的 YOUR_FORMSPREE_ID
- 必须修改 YOUR_GITHUB_USERNAME
- 否则表单无法正常工作！

详细步骤请查看"1-快速开始.txt"

祝部署顺利！🙏
